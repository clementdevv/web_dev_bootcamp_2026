
# mkdir and touch

mkdir -> make directory; creates a new folder
Example:
  mkdir projects

Create multiple directories:
  mkdir dir1 dir2 dir3

touch -> creates an empty file
Example:
  touch file.txt

Create multiple files:
  touch file1.txt file2.txt

Common workflow:
- mkdir my_project
- cd my_project
- touch index.html

Tip:
Use ls to confirm files or folders were created.
