
# Man Pages and Flags

man -> manual; shows documentation for commands
Example:
  man ls

Man pages describe:
- command usage
- available options (flags)
- examples

Flags
- Flags modify how a command behaves
- Usually start with -

Examples with ls:
- ls -l -> long listing format
- ls -a -> show hidden files
- ls -la -> combine multiple flags

Tip:
Use q to exit a man page.
