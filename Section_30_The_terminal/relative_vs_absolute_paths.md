
# Relative vs Absolute Paths

Paths tell the system where a file or directory is located.

Absolute Path
- Full path starting from the root directory (/)
- Works from anywhere in the system
- Example:
  /home/user/Documents/file.txt

Relative Path
- Path relative to the current working directory
- Does not start with /
- Example:
  Documents/file.txt

Special Shortcuts
- .  -> current directory
- .. -> parent directory

Examples:
- cd ..        (move up one directory)
- cd ./folder  (enter folder in current directory)
