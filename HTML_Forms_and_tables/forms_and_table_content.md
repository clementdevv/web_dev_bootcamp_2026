
# New HTML Elements learnt
- **<td>** - defines table cell data
- **<tr>** - defines table row
- **<th>** - defines table header
- **<thead>**
- **<tbody>**
- **<tfoot>**

> - The <form> element is also an essential element that represents a document section containing interactive controls   submitting information
> - It has the action attribute which specifies **WHERE** the form data should be sent
> - The method attribute specifies which HTTP method should be used 
> - The <input> element is usually used within forms and is quite common in use as well
> - The <label> element is also a common on used to represent a caption for an item in a user interface
> - The <button> element is a clickable element used to submit html forms

---

- These are the basic elements to be familiar with when getting introduced to working with tables in HTML
